name: doc-authoring
description: Use when updating README, playground docs, or AGENTS.md in vidget.
---

When changing public API, config options, or mount/display behavior, update all of:

1. `README.md`
2. `playground/src/App.vue` (relevant sections)
3. `src/types.ts` JSDoc comments
4. `AGENTS.md` doc table if a new doc file is added

Keep all documentation in English.
